BuildDB Engine

BuildDB Engine is a custom-built database system developed using Python and file handling techniques. It simulates how real DBMS systems work internally including parsing, execution, storage, and transactions.

The system supports SQL-like queries such as CREATE, INSERT, SELECT, UPDATE, DELETE, and transaction operations like BEGIN, COMMIT, and ROLLBACK.
