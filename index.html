BuildDB Engine

BuildDB Engine is a custom-built database management system developed using Python and file handling techniques. The primary goal of this project is to replicate the core functionalities of a real-world DBMS such as query parsing, execution, storage management, and transaction handling without using any external database libraries.

This system supports SQL-like commands including CREATE, INSERT, SELECT, UPDATE, and DELETE, along with transaction operations like BEGIN, COMMIT, and ROLLBACK. It provides a simplified yet effective understanding of how databases internally process queries and manage data.
